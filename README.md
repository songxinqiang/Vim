#VIM插件集合
仓库将常用插件或插件的github地址关联在一起。
##使用

``` bash
 git clone https://github.com/tqcenglish/Vim ~/.vim
	 cp ~/.vim/vimrc ~/.vimrc
	 git submodule foreach git pull origin master //更新子模块
```
